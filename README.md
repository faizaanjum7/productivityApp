# 🌙 skylar productivity app

a smart productivity app that generates flexible or structured workflows based on user tasks, combining time management with ai-assisted development.

🔗 live: https://skylarfocus-app.vercel.app/

## ✨ technologies

- react
- typescript  
- css  
- google ai studio (ai-assisted development)  
- local storage  

## 🚀 features

- generate flexible or structured task workflows  
- flexible mode: assigns time using pomodoro/session logic  
- structured mode: creates time-based schedules (e.g., 12:30–1:30 coding)  
- task input system for dynamic planning  
- local storage for saving user data  
- customizable user name, theme color, and motivational messages  
- clean and interactive ui

## 📍 the process

this project started as an experiment while exploring google ai studio and vibe coding. the idea was to go beyond a basic to-do app and create something that actually helps users structure their time.

i built the core idea and flow, then used ai tools to accelerate development, followed by making my own tweaks and improvements to refine the experience.

the focus was on giving users two approaches:
- a flexible flow using pomodoro-style sessions  
- a structured flow with fixed time blocks  

this helped explore how different users prefer to plan and manage their work.

## 📚 what i learned

- working with ai tools for faster prototyping and development  
- translating ideas into structured user flows  
- designing flexible vs structured productivity systems  
- using local storage for persistence  
- refining ai-generated output with manual improvements  

## 💭 improvements

- add notifications/reminders for sessions  
- sync data across devices  
- improve flow generation logic  
- add analytics for productivity tracking
